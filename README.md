# easyGDS Marketing Agent

An OpenCode skill for creating on-brand marketing and sales materials for easyGDS travel commerce platform.

## What It Does

Generates professional marketing content including:
- Sales emails (airline executives, travel agents, loyalty programs, tourist boards)
- Landing pages with CTAs
- LinkedIn posts and social content
- Pitch decks and proposals
- Website copy

## Knowledge Base

The skill includes comprehensive reference materials:
- Brand guidelines and tone of voice
- Product portfolio (Dynamic Packages, Stopover, Loyalty, etc.)
- Company information and testimonials (Etihad, Malaysia Airlines, Amadeus)
- Tripovo inventory data (1M+ hotels, 950+ airlines, 250k activities)

## Installation for Local Agents

To use this skill with your local OpenCode agent:

1. Copy the skill directory:
```bash
cp -r .claude/skills/easygds-marketing-skill ~/.claude/skills/
```

2. Verify installation:
```bash
ls ~/.claude/skills/
# You should see: easygds-marketing-skill
```

3. Restart your OpenCode agent to load the skill

The skill will now be available whenever you create marketing content for easyGDS.

## Usage

Simply ask the agent to create marketing materials, and it will automatically reference the brand guidelines and product information to ensure on-brand content.

Example: "Create a sales email to the CEO of an airline about our holiday platform"
